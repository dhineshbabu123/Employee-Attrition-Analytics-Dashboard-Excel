# Employee Attrition Analytics Dashboard Using Advanced Excel
## Project Overview
This project focuses on analyzing employee attrition patterns using Microsoft Excel. The dashboard provides interactive visualizations and key performance indicators (KPIs) that help identify workforce trends and factors influencing employee turnover.
The objective of this project is to support HR teams and management in making data-driven decisions to improve employee retention.

## Key Findings

### Overall Attrition Rate
* The overall attrition rate is 19.97%.

### Department with Highest Attrition
* Finance department shows the highest attrition rate.

### Age Group with Highest Attrition
* Employees aged 18–25 have the highest turnover.

### Job Role with Highest Attrition
* Assistant role records the highest attrition.

### Overtime Impact
* Overtime has minimal impact on employee attrition.

### Salary Impact
* Salary levels show little influence on employee turnover.
